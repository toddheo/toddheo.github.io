## A.1 아임인 서비스의 법적 성격

아임인은 **전자금융거래법 및 관련 금융법규 내에서 운영되는 적법한 전자금융 서비스**입니다.

| 법적 분류 | 세부 내용 |
| --------- | --------- |
| **서비스 유형** | 전자금융업 (플랫폼 기반 금융 중개 서비스) |
| **주요 근거 법령** | 전자금융거래법, 개인정보 보호법, 신용정보의 이용 및 보호에 관한 법률 |
| **금융기관 연계** | 신한은행, 케이뱅크와의 공식 파트너십을 통한 제도권 금융 내 운영 |

## A.2 군 복무 중 금융 서비스 이용 관련 검토

### 현행 군 관련 법규와의 관계

| 검토 항목 | 현행 규정 | 아임인 이용 가능 여부 |
| --------- | --------- | :------------------: |
| **스마트폰 이용** | 부대별 허용 시간·범위 내 사용 가능 | 허용 부대에서 이용 가능 ✅ |
| **금융 거래 금지 여부** | 특별한 금지 규정 없음 | 이용 가능 ✅ |
| **개인 금융 정보 보호** | 개인정보 보호법 동일 적용 | 완전 준수 ✅ |
| **사금융 관련** | 군내 사금융 금지 | 아임인은 제도권 금융 서비스로 해당 없음 ✅ |

> 아임인은 군 내에서 별도의 법적 문제 없이 이용 가능합니다.

## A.4 군 도입 관련 제도적 절차

아임인 군 도입을 위해 거쳐야 하는 제도적 절차는 다음과 같습니다.

| 절차 | 담당 기관 | 소요 시간 (추정) |
| ---- | --------- | :--------------: |
| 도입 정책 결정 | 국방부 | 1~2개월 |
| 업무협약(MOU) 체결 | 국방부 ↔ (주)티웨이브 | 1개월 |
| 군 보안 심의 | 각 군 보안 담당 | 1~2개월 |
| 앱 화이트리스트 등록 | 각 부대 | 1개월 |
| **총 소요 기간** | | **약 3~4개월** |

> (주)티웨이브는 위 모든 절차에 적극 협력하며, 필요한 자료와 기술적 지원을 즉시 제공할 준비가 되어 있습니다.

| 잠재 리스크 | 검토 결과 | 대응 방안 |

| 금융 사고 책임 | 안심서비스로 참여자 보호 | MOU에 책임 범위 명확화 |
