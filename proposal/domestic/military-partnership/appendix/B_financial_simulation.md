| ---- | :--: |
| **참여 장병 수** | 30만 명 |
| **월 평균 납입액** | 20만 원 |

## B.4 국방부 투자 대비 효과(ROI) 분석

### 투입 비용 추정

| 비용 항목 | 연간 추정 비용 | 비고 |
| --------- | :------------: | ---- |
| 금융 교육 콘텐츠 개발 (초기 1회) | 약 1~2억 원 | 초기 비용 |
| 부대 담당관 교육 | 약 5,000만 원/년 | 운영 비용 |
| 아임인 서비스 이용료 | 별도 협의 | 장병 1인당 소액 또는 무료 기준 협의 |
| **합계** | **약 2~3억 원 (초기)** + **약 5천만 원/년 (운영)** | |

### 창출되는 편익 추정

| 편익 항목 | 연간 추정 편익 | 산출 근거 |
| --------- | :------------: | --------- |
| 금융 사고 관련 징계 행정 비용 절감 | 상당 규모 | 징계 건수 20% 감소 가정 |
| 장병 복지 상담 비용 절감 | 일정 규모 | 재무 스트레스 감소 효과 |
| 전역 장병 청년 복지 수요 감소 | 사회적 편익 | 경제적 자립 향상 |
| **무형 편익: 국방부 이미지 제고** | 정책 가치 | 장병 복지 혁신 선도 이미지 |

> **종합 판단:** 투입 비용 대비 창출되는 유·무형 편익이 압도적으로 크며, 이는 본 제안이 국방부 입장에서 매우 합리적인 정책 투자임을 의미합니다.
