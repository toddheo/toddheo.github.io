# 3.2 군 장병 맞춤형 활용 시나리오

**[2026-05-08]**

---

## 3.2.1 복무 일정과 아임인 스테이지의 최적 연계

육군 기준 일반 복무 기간은 18개월입니다. 이 18개월을 아임인 스테이지와 전략적으로 연계하면, 단 한 번의 복무 기간 동안 **복수의 재무 목표를 동시에 달성**할 수 있습니다.

### 18개월 복무 기간 맞춤 저축 설계 (예시)

```
입대(0개월) ─────────────────────────────────── 전역(18개월)
     │                                                │
     │   [스테이지 A] 5개월 → 첫 휴가비               │
     ├───────────────┤                                │
     │               │                                │
     │   [스테이지 B] ──────────── 7개월 → 여름 여행비  │
     ├──────────────────────────┤                     │
     │                          │                     │
     │   [스테이지 C] ──────────────── 9개월 → 연말 목돈│
     ├──────────────────────────────────┤             │
     │                                  │             │
     │   [스테이지 D] ──────────────────────── 11개월  │
     ├────────────────────────────────────────┤       │
     │                                        │       │
     │   [스테이지 E] ─────────────────────────────── 전역 시드머니
     └──────────────────────────────────────────────────┤
                                                        │
                                                전역 자금 수령 완료
```

## 3.2.2 장병 유형별 맞춤 시나리오 4가지

### 시나리오 ① 소액 시작형 (입대 직후, 금융 경험 전무)

- **대상:** 금융 상품을 처음 접하는 병사, 월 급여가 적은 이병~일병 초반
- **추천 전략:** 월 10만 원으로 5개월 단기 스테이지 시작

| 항목 | 내용 |
| ---- | ---- |
| 스테이지 기간 | 5개월 |
| 월 납입액 | 10만 원 |
| 목표 수령액 | 50만 원 |
| 목표 활용처 | 첫 정기 휴가 비용 |
| 기대 효과 | 첫 목돈 달성 경험 → 저축 습관의 씨앗 형성 |

**실제 적용 후기 (유사 사례):**
> *"대학생이라 아르바이트 하거나 용돈 받는게 전부라 큰돈은 없지만 조그만 금액도 아임인 할 수 있어서 좋았어요. 처음엔 돈이 묶이진 않을까 걱정 되었는데 그런 걱정은 안해도 되겠더라고요."* — 아임인 실제 이용자 후기

---

### 시나리오 ② 목표 달성형 (상병~병장, 특정 목표 보유)

- **대상:** 전역 후 구체적인 목표(첫 차, 해외여행, 취업 준비 자금)가 있는 장병
- **추천 전략:** 목표별 복수 스테이지 동시 운용

| 목표 | 스테이지 기간 | 월 납입액 | 목표 수령액 |
| ---- | :----------: | :------: | :---------: |
| 전역폰 구매 | 7개월 | 15만 원 | 105만 원 |
| 전역 여행 | 9개월 | 20만 원 | 180만 원 |
| 취업 준비 자금 | 11개월 | 30만 원 | 330만 원 |
| **합계** | (동시 진행) | **65만 원/월** | **615만 원** |

> 병장 기준 급여 150만 원의 약 43%를 저축하면서도 복무 중 3가지 목표를 달성하는 전략입니다.

---

### 시나리오 ③ 가족 부양형 (경제적 취약 가정 출신 장병)

- **대상:** 가족에게 생활비를 지원해야 하거나, 군 복무 전부터 경제적 어려움을 겪은 장병
- **추천 전략:** 소액 장기 스테이지로 전역 후 가족 지원 자금 마련

| 항목 | 내용 |
| ---- | ---- |
| 스테이지 기간 | 13개월 (최장 스테이지) |
| 월 납입액 | 20만 원 |
| 목표 수령액 | 260만 원 |
| 목표 활용처 | 전역 후 가족 생활비 지원 + 취업 준비 |
| 심리적 효과 | "나도 가족에게 도움이 될 수 있다"는 희망 부여 |

**이 시나리오의 극단적 중요성:**

경제적으로 절망적인 상황에서 군 복무 중 희망을 잃은 장병은 극단적 선택을 고려하는 경우가 있습니다. 아임인은 작은 저축이 쌓여가는 것을 눈으로 확인하게 함으로써, 이 장병들에게 "버틸 이유"를 제공합니다.

---

### 시나리오 ④ 재무 성장형 (적극적 자산 형성 희망 장병)

- **대상:** 전역 후 빠른 경제적 자립을 목표로 하는 의욕적인 병장
- **추천 전략:** 최대 납입액으로 다수 스테이지 동시 운용

| 스테이지 | 기간 | 월 납입액 | 수령 시기 | 수령액 |
| -------- | :--: | :------: | --------- | :----: |
| A | 7개월 | 40만 원 | 복무 7개월 후 | 280만 원 |
| B | 11개월 | 50만 원 | 복무 11개월 후 | 550만 원 |
| C | 13개월 | 60만 원 | 전역 시 | 780만 원 |
| **합계** | | **150만 원/월** | | **1,610만 원** |

> 병장 기준 급여 전액을 저축에 투입하는 최적화 전략으로, 전역 시 약 1,600만 원에 달하는 시드머니 확보가 이론적으로 가능합니다.

## 3.2.3 복무 단계별 금융 여정 설계

| 복무 단계 | 기간 | 재무 목표 | 아임인 활용 방법 |
| --------- | :--: | --------- | --------------- |
| **훈련소·자대 배치** | 0~2개월 | 서비스 이해, 목표 설정 | 가입 완료, 첫 소액 스테이지 시작 |
| **초반 복무** | 3~6개월 | 소액 성공 경험 축적 | 첫 목돈 수령, 두 번째 스테이지 시작 |
| **중반 복무** | 7~12개월 | 복수 스테이지 운용 | 2~3개 스테이지 동시 진행 |
| **후반 복무** | 13~18개월 | 전역 자금 극대화 | 전역 시드머니 스테이지 최우선 유지 |
| **전역 후** | 전역 이후 | 사회 복귀 자금 활용 | 수령 자금으로 첫 사회 생활 안정 진입 |
