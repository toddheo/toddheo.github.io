# 1.3 현행 군 금융 제도의 구조적 한계

**[2026-05-08]**

---

## 1.3.1 현행 군 금융 제도 개요

현재 장병들이 이용 가능한 주요 공식 금융 상품은 크게 세 가지입니다. 각 상품은 나름의 가치를 지니고 있으나, 급변한 급여 수준과 장병들의 다양화된 재무 목표를 충족하기에는 구조적 한계가 명확합니다.

---

## 1.3.2 상품별 심층 분석

### ① 장병내일준비적금

국방부와 시중 은행(신한·하나·기업·농협 등)이 공동으로 운영하는 정책 금융 상품입니다.

| 항목 | 내용 |
| ---- | ---- |
| **금리** | 연 최대 5~6% (정부 지원 이자 포함, 비과세) |
| **납입 한도** | 월 최대 40만 원 |
| **가입 대상** | 현역 병사 전원 |
| **만기** | 전역 시 (복무 기간 내 중도 해지 시 일부 이자 손실) |
| **유동성** | **전무** — 전역 전 인출 불가 |

**구조적 한계:**
- 월 40만 원 납입 한도로는 증가한 급여의 일부만 운용 가능
- 전역 후 일시 수령이라는 단일 결과만 존재 → 복무 중 '목적별 저축(휴가비, 전역폰, 자격증 취득비 등)' 불가능
- 만기 전 인출 시 정부 지원 이자 소멸 → 사실상 중도 해지 억제 구조

---

### ② 군인 공제회

군인 공제회법에 근거하여 운영되는 군인 복지 기관입니다.

| 항목 | 내용 |
| ---- | ---- |
| **납입 방식** | 급여에서 자동 공제 |
| **수익률** | 약 4~5% 수준 (연도별 변동) |
| **가입 대상** | 주로 직업 군인(장교·부사관) |
| **만기** | 전역 또는 퇴역 시 |
| **유동성** | **전무** — 장기 적립형 |

**구조적 한계:**
- 병사(의무 복무)보다 직업 군인 중심 설계로, 18~21개월 복무 병사의 단기 수요와 부합하지 않음
- 단순 적립 외 재무 목표 설정·관리 기능 부재

---

### ③ 일반 시중 예금·적금

병사들이 개인적으로 은행 앱을 통해 이용하는 일반 금융 상품입니다.

| 항목 | 내용 |
| ---- | ---- |
| **금리** | 연 3~4% 수준 (2026년 기준) |
| **유동성** | 만기 전 해지 시 약정 이자 손실 |
| **목표 관리** | 기능 없음 (단순 적립) |
| **소셜 기능** | 없음 |

**구조적 한계:**
- 금리가 장병내일준비적금 대비 낮아 실질적 경쟁력 부재
- 자동 이체 설정이 어려운 병영 환경에서 꾸준한 저축 유지 어려움

---

## 1.3.3 현행 제도의 종합 비교 분석

| 비교 항목 | 장병내일준비적금 | 군인공제회 | 일반 예적금 | **아임인** |
| :-------: | :-------------: | :--------: | :---------: | :--------: |
| **금리 수준** | 최대 6% (우수) | 4~5% | 3~4% | 동등 이상 |
| **유동성** | ❌ 전역 후 수령 | ❌ 전역 후 수령 | △ 중도 해지 시 손실 | ✅ 목표 달성 즉시 수령 |
| **목적별 분산 운용** | ❌ 불가 | ❌ 불가 | ❌ 단일 적립 | ✅ 복수 목표 동시 운용 |
| **소셜 동기부여** | ❌ 없음 | ❌ 없음 | ❌ 없음 | ✅ 그룹 참여, 공동 달성 |
| **단기 목표 저축** | ❌ 불가 | ❌ 불가 | △ 제한적 | ✅ 1~12개월 단위 설정 |
| **금융 교육 효과** | 낮음 | 낮음 | 없음 | **높음** (자기주도 재무 경험) |
| **가입 연령 적합성** | 보통 | 낮음 (직업군인 중심) | 보통 | **높음** (20대 초반 최적화) |
| **사고 예방 기능** | ❌ 없음 | ❌ 없음 | ❌ 없음 | ✅ 건전 저축 습관으로 투기 대체 |

## 1.3.4 구조적 공백의 시각화

현행 군 금융 제도가 커버하지 못하는 영역을 도식화하면 아래와 같습니다.

```
저위험 ◄──────────────────────────────────────────► 고위험
         │                              │
    [예·적금 구역]          [투기 구역]
    장병내일준비적금        주식·코인 투자
    군인공제회             불법 온라인 도박
         │
         │  ← 이 사이의 거대한 공백 →  │
         │                              │
         │    안전성 + 유동성 + 목적성   │
         │    → 현재 아무 상품도 없음    │
         │                              │
    ─────────────────────────────────────────────────
    장기 적립(5~10년)   중단기(6개월~2년)   단기 투기
    전역 후 수령        복무 중 필요        즉각 수익 기대
```

> **이 공백이 바로 장병들을 투기 시장으로 내모는 근본 원인입니다.**
> 아임인은 이 공백을 정확히 채웁니다.

## 1.3.5 장병이 원하는 것 vs. 현행 제도가 주는 것

장병들이 금융 상품을 통해 실질적으로 원하는 것은 단순한 '이자 수익'이 아닙니다. 아래 표는 장병들의 실제 재무 니즈와 현행 제도의 간극을 구체적으로 보여줍니다.

| 장병의 실제 니즈 | 현행 제도 충족 여부 | 아임인 충족 여부 |
| --------------- | :----------------: | :-------------: |
| 복무 중 휴가비 자금 마련 | ❌ | ✅ |
| 전역 후 이직·취업 준비 자금 | △ (전역 후 수령 가능) | ✅ |
| 부모님께 효도하고 싶은 욕구 충족 | ❌ | ✅ |
| 전역 후 첫 자동차·전역폰 구매 | ❌ | ✅ |
| "내가 스스로 돈을 모았다"는 성취감 | ❌ | ✅ |
| 친구나 가족과 함께하는 재미 | ❌ | ✅ |
| 만약의 사태 대비 비상금 확보 | ❌ | ✅ |
| 재무적 자신감·독립심 형성 | ❌ | ✅ |

> 현행 군 금융 제도는 '저축 금액의 보관'이라는 수동적 기능에 머물러 있습니다. 아임인은 장병들이 **능동적으로 재무 목표를 설정하고 달성하는 주체**가 될 수 있도록 설계되었습니다.
