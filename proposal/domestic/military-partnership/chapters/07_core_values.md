# 2.3 핵심 제공 가치: 안전성·유동성·소셜 동기부여의 3축 결합

**[2026-05-08]**

---

## 2.3.1 3축 핵심 가치 개요

아임인의 핵심 가치는 단일 기능이 아닌, 세 가지 요소가 **유기적으로 결합**될 때 완성됩니다. 이 결합 방식은 경쟁 상품이 단기간에 모방하기 어려운 아임인만의 구조적 강점입니다.

```
         안전성 (Safety)
              ▲
             / \
            /   \
           /     \
          /  아임인 \
         /           \
        ▼─────────────▼
  유동성          소셜 동기부여
(Liquidity)      (Social Motivation)

※ 세 꼭짓점이 동시에 충족될 때만 '진정한 장병 재무 솔루션'이 완성됩니다.
```

---

## 2.3.2 제1축: 안전성 (Safety)

### 안전성의 정의

아임인에서 '안전성'은 단순히 원금이 보호된다는 의미를 넘어, **장병이 예상치 못한 상황에서도 손실 없이 자금을 회수할 수 있다는 구조적 보장**을 의미합니다.

### 안전성을 뒷받침하는 3가지 장치

**장치 ① KCB 기반 신용평가 시스템**

- 아임인 참여자 전원에 대해 코리아크레딧뷰로(KCB)와 연계한 신용평가를 실시합니다.
- 신용 등급 기준 이하의 고위험 참여자는 그룹 구성에서 사전 분리됩니다.
- 결과: 그룹 내 부실 참여자 발생 확률이 구조적으로 낮습니다.

**장치 ② 안심서비스 (연체 보호 메커니즘)**

- 참여자가 연체 발생 시 두 가지 옵션 중 선택 가능:
  - **옵션 A:** 연체 즉시 원금의 98% 수령 (선지급)
  - **옵션 B:** 연체 해소 후 원금 100% + 지연 이자 수령
- 이 구조 덕분에 타인의 연체로 인한 피해가 사실상 차단됩니다.

**장치 ③ 제1금융권 수준 연체율**

- 2026년 4월 기준 아임인의 연체율: **0.22%**
- 국내 주요 시중은행 가계대출 연체율(1~2% 수준) 대비 현저히 낮은 수치
- 8년간 60만 명 이상이 참여한 플랫폼에서 달성한 실적으로, 단순 홍보가 아닌 **검증된 리스크 관리 역량**

### 군 장병 안전성 적용 의미

장병들은 투자나 저축에서 '혹시 내 돈이 사라지면 어떡하지?'라는 불안을 가장 크게 느낍니다. 아임인의 안심서비스와 낮은 연체율은 이 불안을 제도적으로 해소하여, 장병이 안심하고 저축에 집중할 수 있도록 합니다.

## 2.3.3 제2축: 유동성 (Liquidity)

### 유동성의 정의

아임인에서 '유동성'은 **장병이 목표로 설정한 시점에 정확히 자금을 수령할 수 있다는 계획성**을 의미합니다. 기존 예적금처럼 '전역 후에나 받을 수 있는 돈'이 아니라, 복무 기간 중 원하는 목적에 맞춰 자금을 활용할 수 있습니다.

### 군 장병 유동성 활용 시나리오

| 목표 | 스테이지 기간 | 월 납입액 (예시) | 수령 시기 |
| ---- | :-----------: | :--------------: | --------- |
| 첫 정기 휴가비 | 5개월 | 20만 원/월 | 입대 5개월 후 |
| 여름/겨울 휴가 여행비 | 7개월 | 30만 원/월 | 7개월 후 |
| 전역폰 구매 자금 | 11개월 | 40만 원/월 | 복무 11개월 후 |
| 전역 후 시드머니 | 13개월 | 50만 원/월 | 전역 시 |

> 장병들은 복수의 스테이지에 동시 참여하여 **여러 목표를 동시에 달성**할 수 있습니다. 이것이 기존 단일 예금과 근본적으로 다른 점입니다.

### 유동성이 투기 억제에 기여하는 원리

장병들이 투기적 투자에 끌리는 이유 중 하나는 '예적금은 묶여있어 답답하고, 그렇다고 장기간 인내할 자신도 없다'는 심리입니다. 아임인은 이 갈증을 **단기 목표 달성이라는 건전한 방식으로 해소**합니다.

---

## 2.3.4 제3축: 소셜 동기부여 (Social Motivation)

### 소셜 동기부여의 정의

아임인의 핵심 차별점 중 하나는 **혼자가 아닌 그룹으로 함께 저축한다는 소셜 구조**입니다. 이는 개인 예금이나 적금에서 얻을 수 없는 강력한 지속 동기를 제공합니다.

### 소셜 구조가 만들어내는 심리적 효과

| 효과 | 설명 |
| ---- | ---- |
| **상호 책임감** | 그룹 구성원 모두가 영향을 받는 구조상, 개인 이탈에 대한 책임감 형성 |
| **경쟁적 저축** | 같은 목표를 가진 동료들과의 긍정적 경쟁 심리가 꾸준한 납입을 유도 |
| **성취 공유** | 목표 달성 후 그룹 내 축하 공유 → 긍정적 경험의 반복 확인 |
| **또래 효과 전환** | 부정적 또래 동조(빚투 따라하기) → 긍정적 또래 동조(함께 저축) |

### 군 병영 환경에서의 소셜 효과 배가 요인

군 병영은 세상에서 가장 강한 '또래 집단' 환경입니다. 동일한 공간, 동일한 시간을 공유하는 병사들 사이에서 소셜 저축 경험은 더욱 강력하게 작동합니다.

- "같은 생활관 병사들이 함께 목돈 모으기" → 자연스러운 건전 저축 문화 형성
- "먼저 전역한 선임이 아임인으로 전역 자금 모았다" → 선임→후임 전파 구조
- 금융 사고의 전파 경로(또래 동조)가 **건전 저축 전파 경로로 전환**

---

## 2.3.5 3축 결합의 시너지 효과

각 축이 개별적으로도 가치가 있지만, 세 축이 **동시에 작동할 때** 단순 합산 이상의 시너지가 발생합니다.

```
안전성 단독:    "안전하지만 지루하다" → 장기 지속 어려움
유동성 단독:    "인출은 쉽지만 목적 없이 쓰게 된다" → 저축 효과 없음
소셜 단독:      "함께하지만 위험하다" → 집단 투기로 변질 가능

3축 결합 시:
  → "안전하게 + 목표 시점에 받을 수 있고 + 함께한다"
  → 진정한 재무 목표 달성 + 건전 저축 습관 형성
  → 군내 금융 사고의 구조적 예방
```
