# 4.4 전군 확대 방안 및 지속 운영 체계

**[2026-05-08]**

---

## 4.4.1 확대 원칙: 검증 후 단계적 확장

전군 확대는 파일럿 성과 데이터를 기반으로 신중하게 진행합니다. 성급한 확대보다는 검증된 방식으로 안정적인 확대를 추구합니다.

---

## 4.4.2 중규모 확대 (전군 확대 전 단계)

파일럿 성과 검증 후, 5개 사단으로 확대 운영합니다.

| 확대 항목 | 내용 |
| --------- | ---- |
| **대상 부대** | 육·해·공군·해병대 각 1개 사단 및 유사 규모 부대 |
| **대상 장병 수** | 약 1만~1만 5천 명 |
| **목표 가입자** | 6,000명 이상 (40% 가입률) |
| **파일럿 대비 개선 사항** | 파일럿에서 도출된 개선 의견 전면 반영 |

### 군종별 특화 접근 방안

| 군종 | 특수 환경 | 맞춤 대응 방안 |
| :--: | --------- | -------------- |
| **육군** | 훈련·전투 부대, 복무 18개월 | 5·7·9개월 단기 스테이지 집중 |
| **해군** | 함정 근무, 장기 출항 | 자동 납입 기능 강화, 출항 중 알림 유예 기능 |
| **공군** | 상대적 인터넷 환경 양호 | 앱 기능 최대 활용, 커뮤니티 기능 적극 사용 |
| **해병대** | 고강도 훈련, 강한 동료 의식 | 소셜 기능 강조, 생활관 단위 그룹 참여 유도 |

## 4.4.3 전군 도입 방안 (최종 목표)

### 도입 목표

| 항목 | 목표 |
| ---- | ---- |
| **대상** | 현역 병사 전원 (약 50만 명) |
| **연간 신규 가입률 목표** | 입대 장병의 70% 이상 |
| **안정적 이용자 규모 (3년차)** | 약 30만 명 이상 |

### 전군 도입 지원 체계

**① 부대별 담당관 네트워크 구축**
- 전국 모든 부대(대대급 이상)에 아임인 담당관 1명 지정
- 담당관 교육을 위한 연 2회 전국 교육 프로그램 운영
- 담당관 간 온라인 네트워크 및 우수 사례 공유 플랫폼 운영

**② 금융 교육과의 통합 운영**
- 입대 후 필수 교육(기초 군사훈련) 과정에 '재무 건전성 교육' 모듈 포함
- 아임인 가입을 교육 수료 직후 자연스럽게 연결하는 동선 설계

**③ 선임-후임 전파 체계**
- 아임인 이용 중인 선임 장병이 후임 장병에게 자연스럽게 소개하는 문화 조성
- 우수 절약·저축 경험 장병을 '재무 멘토'로 지정하는 제도 운영 (국군복지단 협력)

## 4.4.4 지속 운영 체계

전군 도입 이후에도 서비스 품질과 효과를 유지하기 위한 지속 운영 체계를 구성합니다.

### 정기 성과 평가 체계

| 평가 항목 | 주기 | 담당 기관 | 활용 방법 |
| --------- | :--: | --------- | --------- |
| 이용자 만족도 조사 | 분기 | 국군복지단 | 서비스 개선 반영 |
| 금융 사고 현황 분석 | 반기 | 각 군 | 효과 측정 및 보완 |
| 서비스 성과 종합 평가 | 연 1회 | 국방부 주관 | 지속 운영 여부 결정 |
| 기술 보안 점검 | 연 2회 | (주)티웨이브 | 보안 취약점 사전 제거 |

### 서비스 고도화 로드맵

아임인은 군 도입 이후에도 지속적으로 군 장병 맞춤형 기능을 발전시킵니다.

| 시기 | 고도화 내용 |
| :--: | ----------- |
| **도입 1년차** | 군 장병 전용 금융 교육 콘텐츠 확충 |
| **도입 2년차** | 전역 예정 장병 대상 사회 복귀 재무 설계 서비스 추가 |
| **도입 3년차** | 전역 후에도 지속 이용 가능한 민간 연계 서비스 확대 |
