# 3.1 서비스 개요 및 운용 방식

**[2026-05-08]**

---

## 3.1.1 아임인(imin)이란

아임인은 대한민국의 전통 공동체 금융 문화인 '계(契)' 모임의 원리를 현대적으로 디지털화한 **소셜 저축 금융 플랫폼**입니다. 단순한 적금 앱이 아니라, 공동의 목표를 가진 사람들이 함께 저축하고 함께 목돈을 나누는 경험을 제공하는 새로운 범주의 금융 서비스입니다.

- **서비스명:** 아임인(imin)
- **운영사:** (주)티웨이브 (TWAVE Inc.)
- **설립 연도:** 2016년
- **서비스 출시:** 2016년 (이후 지속적 기능 개선 중)
- **현황 (2026년 4월 기준):** 누적 거래액 약 3조 863억 원, 총 참여 인원 60만 7,169명

---

## 3.1.2 핵심 운용 원리: 디지털 '계모임'

### 전통 계모임의 디지털 혁신

전통 계모임은 수백 년간 한국 사회에서 검증된 공동체 금융 모델입니다. 참여자들이 매월 일정 금액을 납입하고, 순번에 따라 전체 금액을 수령하는 방식입니다. 아임인은 이 원리를 디지털화하여 다음과 같이 발전시켰습니다.

| 항목 | 전통 계모임 | 아임인 |
| ---- | :--------: | :----: |
| **참여 방식** | 지인 기반, 대면 모집 | 플랫폼 자동 매칭, 비대면 |
| **신뢰 보장** | 인적 신뢰 | KCB 신용평가 + 안심서비스 |
| **투명성** | 계주 의존 | 앱 내 실시간 공개 |
| **연체 처리** | 계주 판단, 분쟁 발생 | 안심서비스로 자동 처리 |
| **접근성** | 지인 네트워크 필요 | 앱 설치만으로 즉시 참여 가능 |
| **규모** | 소규모 한정 | 전국 60만 7천 명+ |

### 스테이지 운용 방식 (기본 구조)

```
예시: 9명 그룹, 월 30만 원 납입, 9개월 스테이지

    1회차: [참여자 A 수령] 300만 원
           모든 참여자 30만 원 납입 → A가 300만 원 수령
    │
    2회차: [참여자 B 수령] 300만 원
    │
    3회차: [참여자 C 수령] 300만 원
    ...
    │
    9회차: [참여자 I 수령] 270만 원
    └→ 스테이지 완료: 모든 참여자가 목돈 270만 원 달성
```

> 핵심: 어느 순번이든 납입 총액(30만 원 × 9개월 = 270만 원)은 동일합니다. 선순번 수령자는 미래에 납부할 돈을 먼저 받는 방식이며, 후순번 수령자는 시간이 지나면 반드시 목돈을 수령하게 됩니다.

## 3.1.3 주요 기능 상세

### ① 스테이지 선택 및 참여

| 구분 | 내용 |
| ---- | ---- |
| **스테이지 기간** | 5·7·9·11·13개월 (홀수 순번계) |
| **납입 금액** | 월 10만 원 ~ 최대 160만 원 |
| **복수 참여** | 여러 스테이지에 동시 참여하여 복수 목표 동시 진행 가능 |
| **자동 납입** | 매월 지정일 자동 이체로 별도 관리 불필요 |

### ② 진행 상황 실시간 확인

- 현재 스테이지의 납입 회차, 잔여 기간, 예상 수령 금액을 앱 메인화면에서 즉시 확인
- 목표 달성까지의 진행률 시각화로 지속적 동기 부여

### ③ 커뮤니티 기능

- 같은 스테이지 참여자 간 커뮤니티 채널 운영
- 목돈 수령 후기 공유, 목표 달성 인증 등 긍정적 경험 전파
- 아임인 전체 사용자 커뮤니티를 통한 재테크 정보 교류

### ④ 안심서비스 및 고객 보호

- 연체 발생 즉시 자동 처리
- 참여자의 손실 최소화를 위한 이중 안전장치 작동
- 24시간 고객 지원 채널 운영

## 3.1.4 기술 인프라 및 보안 체계

아임인은 금융 서비스로서 최고 수준의 기술 보안을 유지합니다.

| 보안 항목 | 적용 기술 |
| --------- | --------- |
| **데이터 암호화** | 금융 정보 AES-256 암호화 저장 및 전송 |
| **본인인증** | 금융실명법 준수 실명 확인 + 생체인증 지원 |
| **신용정보 보호** | KCB 연동 시 개인신용정보 보호법 완전 준수 |
| **계좌 연동** | 오픈뱅킹 표준 API를 통한 안전한 계좌 연결 |
| **이상거래 탐지** | 24시간 FDS(이상금융거래탐지시스템) 운영 |
