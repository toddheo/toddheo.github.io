# 2.1 도입 목적: 군내 금융 사고 근절과 건전 저축 문화 정착

**[2026-05-08]**

---

## 2.1.1 본 제안의 3대 도입 목적

아임인 군 도입 제안의 핵심 목적은 단순한 금융 상품 소개가 아닙니다. 이는 **장병들의 건전한 재무 습관 형성을 통해 군내 금융 사고를 구조적으로 예방하고, 나아가 전역 후 사회로 복귀하는 청년들의 경제적 자립을 지원하는 국가적 과제**를 수행하는 데 있습니다.

```
┌─────────────────────────────────────────────────────────────────┐
│                    3대 도입 목적                                  │
│                                                                   │
│  목적 1        목적 2         목적 3                              │
│  금융 사고     저축 습관      전역 후                             │
│  예방·근절     형성 지원      경제적 자립                         │
│                                                                   │
│  빚투·도박    자기주도 재무   시드머니 확보                        │
│  근절         목표 달성      사회 복귀 성공                        │
└─────────────────────────────────────────────────────────────────┘
```

---

## 2.1.2 목적 1 — 군내 금융 사고 예방 및 근절

### 목표 정의

복무 중 발생하는 빚투, 불법 도박, 사금융 이용 등 금융 관련 사고를 **구조적으로 차단**합니다. 단순한 교육·홍보가 아닌, 장병들이 자발적으로 건전한 저축에 참여하도록 유도하는 **행동 설계(Behavioral Design)**를 통해 달성합니다.

### 달성 방법

| 방법 | 설명 |
| ---- | ---- |
| **선점 효과** | 급여 입금 직후 아임인 자동 납입 설정으로 투기 자금 원천 차단 |
| **목표 가시화** | 달성 목표와 진행률이 항상 화면에 표시되어 중도 이탈 방지 |
| **소셜 책임감** | 그룹 참여 구조상 개인 이탈이 타인에게 영향을 주는 구조로 자기 통제력 강화 |
| **대체재 제공** | 불법 도박이나 투기가 제공하는 '흥미와 기대감'을 건전한 방식으로 대체 |

---

## 2.1.3 목적 2 — 자기주도적 저축 습관 형성

### 목표 정의

군 복무라는 제한된 환경에서도 장병들이 **능동적으로 재무 목표를 설정하고 달성하는 경험**을 반복하게 합니다. 이 경험은 전역 후에도 지속되는 평생의 금융 역량이 됩니다.

### 습관 형성 3단계 모델 (아임인 기반)

```
[1단계] 목표 설정
  └→ 복무 기간, 목적(휴가비·전역폰·시드머니)에 맞춰 스스로 스테이지 선택

[2단계] 실행 및 진행
  └→ 매월 자동 납입, 진행률 확인, 그룹 내 구성원과 공유

[3단계] 달성 및 재설정
  └→ 목돈 수령 → 성취감 → 새 목표 설정 → 습관화 완성
```

> 실제 사용자 사례에서 확인된 바와 같이, 이 3단계 사이클을 **2~3회 반복하면 금융 습관이 자동화**됩니다. 군 복무 기간(18~21개월) 내에 충분히 3회 이상의 사이클 완성이 가능합니다.

---

## 2.1.4 목적 3 — 전역 후 경제적 자립 기반 조성

### 목표 정의

군 복무가 단순한 의무 이행이 아닌, **경제적 자립을 위한 준비 기간**으로 기능하도록 합니다. 전역 시 일정 수준의 시드머니를 보유한 장병은 사회 진입 장벽이 현저히 낮아집니다.

### 전역 후 효과 시뮬레이션

| 복무 기간 저축 방식 | 전역 시 보유 자금 (추정) | 전역 후 첫 3개월 생활 능력 |
| :----------------: | :---------------------: | :------------------------: |
| 기존 예적금만 이용 | 약 100만~200만 원 | 취약 |
| 아임인 병행 이용 | 약 300만~700만 원 | 안정적 |
| 아임인 적극 활용 | 약 500만~900만 원 | **양호** |

> 전역 직후 경제적 안정감은 무직 상태에서의 충동적 투자나 대출 유혹을 차단하는 데 결정적 역할을 합니다.

## 2.1.5 3대 목적 달성 시 국방부가 얻는 것

아임인 도입이 성공적으로 정착될 경우, 국방부 및 군 기관은 다음과 같은 정책적 성과를 도출할 수 있습니다.

| 성과 항목 | 세부 내용 |
| --------- | --------- |
| **병영 문화 개선** | 금융 사고 감소 → 징계 건수 감소 → 지휘 환경 개선 |
| **장병 복지 수준 향상** | 금전적 스트레스 감소 → 정신건강 및 사기 향상 |
| **전역 후 사회 적응 성공** | 경제적 자립 기반 조성 → 청년 복지 수요 감소 |
| **정책 성과 가시화** | 병사 처우 개선 정책의 실질적 성과로 홍보 가능 |
| **국방부 이미지 제고** | '진취적이고 혁신적인 장병 복지' 정책 선도 이미지 구축 |
