# 4.3 파일럿(시범 운영) 세부 계획

**[2026-05-08]**

---

## 4.3.1 파일럿 운영의 목적

파일럿은 단순한 시험이 아닙니다. 실제 군 환경에서 아임인이 효과적으로 작동함을 **데이터로 증명**하고, 전군 확대를 위한 최적 운영 방안을 도출하는 **검증 과정**입니다.

---

## 4.3.2 파일럿 부대 선정 기준

아래 기준을 충족하는 1개 사단을 파일럿 부대로 선정합니다.

| 선정 기준 | 내용 | 이유 |
| --------- | ---- | ---- |
| **스마트폰 허용 부대** | 영내 스마트폰 사용이 허용된 부대 | 앱 접근성 확보 |
| **규모** | 약 5,000~8,000명 규모의 사단 | 통계적 의미 있는 표본 확보 |
| **지휘관 의지** | 복지 혁신에 적극적인 사단장 부대 | 성공적 운영 위한 내부 추진력 |
| **지역 다양성** | 수도권, 비수도권 중 택일 | 지역 편향 없는 결과 확보 |
| **혼합 계급 구성** | 이병~병장 비율이 고른 부대 | 계급별 효과 비교 가능 |

## 4.3.3 파일럿 운영 세부 절차

### 단계 1: 사전 교육 (파일럿 시작 전 2주)

| 교육 항목 | 대상 | 시간 | 방법 |
| --------- | ---- | :--: | ---- |
| **부대 담당관 교육** | 각 중대 담당관 1명씩 | 4시간 | (주)티웨이브 강사 직접 방문 교육 |
| **장병 기초 금융 교육** | 전체 대상 장병 | 30분 | 영상 콘텐츠 + 담당관 진행 |
| **아임인 앱 가입 안내** | 전체 대상 장병 | 20분 | 단계별 가입 안내 자료 배포 |

### 단계 2: 가입 및 첫 스테이지 시작 (1~2주차)

- 자율 가입 원칙 (강제 가입 없음)
- 가입 장려를 위한 초기 가입 혜택 제공 (국군복지단 협의)
- 담당관을 통한 가입 지원 및 질의응답

### 단계 3: 운영 및 모니터링 (3주~6개월)

| 모니터링 항목 | 주기 | 담당 |
| :----------: | :--: | :--: |
| 가입자 수 현황 | 주 1회 | (주)티웨이브 |
| 스테이지 납입 완료율 | 월 1회 | (주)티웨이브 |
| 장병 만족도 조사 | 월 1회 | 국군복지단 |
| 금융 관련 이슈 발생 여부 | 즉시 보고 | 부대 담당관 |

### 단계 4: 성과 평가 및 보고 (6개월 차)

- 전체 데이터 종합 분석
- 정성적 인터뷰 (가입 장병 20명, 미가입 장병 10명 표본)
- 성과 보고서 작성 및 국방부 제출

## 4.3.4 파일럿 성과 지표 (KPI) 상세

| KPI | 측정 기준 | 목표치 | 측정 방법 |
| :-: | --------- | :----: | --------- |
| **가입률** | 대상 장병 대비 가입 완료 비율 | ≥50% | 앱 데이터 |
| **3개월 유지율** | 가입 후 3개월간 지속 납입 비율 | ≥80% | 앱 데이터 |
| **첫 스테이지 완료율** | 첫 스테이지 만기 수령 비율 | ≥90% | 앱 데이터 |
| **만족도 점수** | 이용자 설문 5점 척도 | ≥4.0 | 설문 조사 |
| **순추천지수(NPS)** | "주변에 추천하겠다" 의향 비율 | ≥50 | 설문 조사 |
| **재참여 의향** | 스테이지 완료 후 재참여 의향 | ≥80% | 설문 조사 |

## 4.3.5 파일럿 예상 성과 및 전군 확대 판단 기준

파일럿 완료 후 다음 기준을 충족하면 전군 확대를 권고합니다.

| 확대 판단 기준 | 내용 |
| ------------- | ---- |
| **가입률 달성** | 목표 가입률(50%) 이상 달성 |
| **안전성 검증** | 파일럿 기간 중 대형 금융 사고 0건 |
| **만족도 달성** | 이용자 만족도 4.0/5.0 이상 |
| **금융 사고 감소** | 대조군 대비 금융 관련 징계 건수 감소 추세 |
| **이용자 추천** | NPS 50 이상 (강한 자발적 추천 의향) |

> **목표:** 파일럿 기간 6개월 내 모든 기준을 달성하여 2027년 하반기 중규모 확대로 이어지도록 합니다.
