# 6.4 주요 언론 보도 (최신순 3건 상세 요약)

**[2026-05-08]**

---

## 6.4.1 보도 개요

아임인과 (주)티웨이브는 국내 주요 언론으로부터 꾸준한 주목을 받아왔습니다. 아래에 소개하는 최신 3건의 보도는 아임인의 현재 상태와 성장성을 가장 정확하게 보여줍니다.

---

## 보도 1: 서울신문 (2026년 3월 24일)

### 제목

**"티웨이브, '아임인' 리브랜딩 단행…사용자 경험 기반 브랜드 전략 강화"**

### 주요 내용 요약

서비스 론칭 10주년을 맞이한 아임인이 '함께', '성장', '습관'이라는 세 가지 핵심 가치를 중심으로 브랜드 전략을 전면 재정비했습니다.

| 리뉴얼 항목 | 내용 |
| ----------- | ---- |
| **핵심 가치 재정립** | 함께(Together) / 성장(Growth) / 습관(Habit) |
| **배경** | 서비스 10주년, 60만 명 이용자 경험을 바탕으로 한 브랜드 고도화 |
| **전략 방향** | 단순 저축 앱을 넘어 '재무 습관 형성 플랫폼'으로의 정체성 강화 |

### 군 도입 관련 시사점

브랜드 리뉴얼의 세 가지 핵심 가치인 '함께', '성장', '습관'은 군 장병들에게 필요한 것과 정확히 일치합니다.
- **함께:** 병영 공동체 문화와의 자연스러운 조화
- **성장:** 복무 기간 중 개인 역량 및 자산 성장
- **습관:** 평생 지속되는 건전 재무 습관 형성

## 보도 2: 세계일보 (2026년 3월 5일)

### 제목

**"티웨이브 '아임인', 2025년 거래액 1조 원 돌파…매출 116억 원 달성"**

### 주요 내용 요약

아임인이 2025년 한 해에만 거래액 1조 원을 돌파하며 핀테크 업계에서 독보적인 성과를 입증했습니다.

| 성과 지표 | 수치 |
| --------- | :--: |
| **2025년 연간 거래액** | 1조 원 이상 |
| **2025년 매출** | 116억 원 |
| **사업 특성** | 흑자 경영, 안정적 성장세 유지 |

### 군 도입 관련 시사점

연간 거래액 1조 원은 아임인이 **대규모 자금을 안전하게 처리할 수 있는 검증된 운영 역량**을 갖추고 있음을 의미합니다. 전군 50만 명 장병 도입 시 추가로 발생할 거래 규모를 충분히 처리할 수 있는 인프라가 이미 갖추어져 있습니다.

## 보도 3: 머니투데이 (2025년 10월 23일)

### 제목

**"티웨이브 '아임인', 거래 건수 280만 건 돌파… 커뮤니티형 핀테크의 성장 입증"**

### 주요 내용 요약

아임인이 누적 거래 건수 280만 건을 달성하며 커뮤니티형 핀테크 모델의 유효성을 재확인했습니다.

| 성과 지표 | 수치 |
| --------- | :--: |
| **누적 거래 건수** | 280만 건 돌파 |
| **누적 거래액 (2025.10)** | 2조 4,000억 원 |
| **연속 성장률** | 8년 연속 평균 250% |

### 8년 연속 250% 성장의 의미

| 연도 | 성장률 | 특이사항 |
| :--: | :----: | -------- |
| 2016~2018 | 250%+ | 초기 폭발 성장 |
| 2019~2021 | 250%+ | 코로나 기간 중에도 성장 유지 |
| 2022~2024 | 250%+ | 급여 인상 시기와 맞물려 잠재 수요 확대 |
| 2025 | 250%+ | 연간 거래액 1조 원 달성 |

> 특히 코로나 팬데믹 기간(2020~2021)에도 성장세를 유지했다는 사실은 아임인의 **서비스 탄력성(Resilience)**을 증명합니다.

### 군 도입 관련 시사점

8년 연속 250% 성장은 단순한 일시적 유행이 아닌 **구조적 수요**에 기반한 성장임을 의미합니다. 군 장병이라는 50만 명의 새로운 수요층과 결합될 경우, 성장세는 더욱 가속화될 것입니다.

## 6.4.2 미디어 포트폴리오 종합

이상 3건의 보도가 전달하는 핵심 메시지는 명확합니다.

| 보도 | 핵심 메시지 |
| ---- | ----------- |
| 서울신문 (2026.03) | 10년의 신뢰를 바탕으로 진화하는 브랜드 |
| 세계일보 (2026.03) | 1조 원 거래·116억 매출의 검증된 사업 모델 |
| 머니투데이 (2025.10) | 8년 연속 성장으로 입증된 커뮤니티 금융의 힘 |

> 세 건 모두 단기 성과가 아닌 **장기 지속 가능한 성장 서비스**임을 일관되게 보도하고 있습니다. 국방부가 장기 파트너십을 맺기에 가장 이상적인 서비스임을 미디어가 독립적으로 검증한 것입니다.
