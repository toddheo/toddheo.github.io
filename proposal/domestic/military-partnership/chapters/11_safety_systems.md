# 3.3 안전성 및 리스크 관리 체계

**[2026-05-08]**

---

## 3.3.1 안전성 설계의 철학

아임인의 안전성 설계는 단 하나의 원칙에서 출발합니다.

> **"참여자의 원금은 어떠한 상황에서도 보호되어야 한다."**

이 원칙을 실현하기 위해, 아임인은 다음과 같이 중첩적인 안전장치를 구축하였습니다.

---

## 3.3.2 안전장치 1: KCB 신용평가 기반 참여자 사전 선별

### 작동 원리

아임인 신규 가입 시, 모든 참여자는 코리아크레딧뷰로(KCB)와 연계한 신용 평가 과정을 거칩니다.

```
신규 가입 신청
      │
      ▼
KCB 신용 조회 (본인 동의 기반)
      │
      ├── [신용 기준 충족] → 스테이지 참여 허용
      │
      └── [신용 기준 미달] → 제한된 스테이지만 참여 가능 또는 참여 제한
```

### 효과

- 연체 가능성이 높은 고위험 참여자가 그룹에 합류하는 것을 사전에 방지
- 건전한 참여자들로 구성된 그룹의 높은 완료율 유지
- 결과: **연체율 0.22% 달성** (제1금융권 시중은행 수준)

---

## 3.3.3 안전장치 2: 안심서비스 (연체 발생 시 채권 보호)

### 작동 원리

아임인의 안심서비스는 특정 참여자가 연체할 경우, **나머지 참여자의 원금이 자동으로 보호**되는 구조입니다.

```
연체 발생
      │
      ▼
아임인 안심서비스 자동 작동
      │
      ├── 옵션 A: 원금의 98% 즉시 수령 (선지급)
      │          └→ 2일 내 지급, 확실한 손실 최소화
      │
      └── 옵션 B: 연체 해소 후 원금 100% + 지연이자 수령
                  └→ 연체 해소 후 이자까지 추가 수령
```

### 군 장병 적용 의미

군 복무 중 어떤 동료 장병이 갑자기 연체하더라도, 나의 납입 자금은 두 가지 방법 중 하나로 반드시 돌려받을 수 있습니다. 불법 도박이나 투기에서 발생하는 '한 방에 전액 손실' 위험과는 구조적으로 다릅니다.

## 3.3.4 안전장치 3: 제도권 금융기관 파트너십

아임인은 신한은행, 케이뱅크와의 공식 파트너십을 통해 **제도권 금융 인프라 위에서 운영**됩니다.

| 항목 | 내용 |
| ---- | ---- |
| **자금 관리** | 파트너 금융기관의 관리 계좌를 통한 자금 운용 |
| **감독 체계** | 금융 당국의 간접적 감독 구조 내 운영 |
| **법규 준수** | 전자금융거래법, 개인정보 보호법 완전 준수 |

---

## 3.3.5 장병 개인정보 보호 체계

군 장병의 개인정보 보호는 일반 이용자보다 더 엄격한 기준을 적용합니다.

| 보호 항목 | 수집 여부 | 처리 방식 |
| :-------: | :-------: | --------- |
| 성명 | 수집 | 금융실명법 기준 본인 확인 용도로만 사용 |
| 연락처 | 수집 | 서비스 알림 발송 용도로만 사용 |
| 계좌번호 | 수집 | 오픈뱅킹 표준 API, 암호화 저장 |
| 신용정보 | 조회 (본인 동의) | 신용 평가 후 즉시 파기, 저장 불가 |
| **군 관련 정보** | **미수집** | 부대명, 계급, 복무 기간 등 군 정보는 일절 수집하지 않습니다 |

> **군사 보안 우려 사전 해소:** 아임인은 금융 서비스로서 군사 관련 정보를 전혀 수집하지 않습니다. 장병의 금융 거래 데이터는 군사 기밀과 완전히 분리 관리됩니다.
