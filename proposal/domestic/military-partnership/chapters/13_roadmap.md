# 4.2 단계별 추진 로드맵

**[2026-05-08]**

---

## 4.2.1 추진 로드맵 전체 조감

아임인 군 도입은 총 4단계로 나누어 신중하고 검증된 방식으로 진행됩니다. 파일럿의 성과가 충분히 검증된 이후에만 다음 단계로 확대함으로써, 국방부의 위험 부담을 최소화합니다.

```
2026년 하반기    2027년 상반기   2027년 하반기   2028년 이후
      │               │               │               │
   [1단계]         [2단계]         [3단계]         [4단계]
   협약·준비        파일럿          확대            전군
   (3개월)        운영·검증      (중규모)          도입
                  (6개월)        (6개월)         (지속운영)
      │               │               │               │
    MOU 체결      1개 사단급      5개 사단급     전군 50만
    기술 개발     2,000명 목표   1만 명 목표    장병 대상
    교육 개발     성과 측정       확대 검증      정착 운영
```

---

## 4.2.2 제1단계: 협약 및 준비 단계 (2026년 하반기, 약 3개월)

### 목표

업무협약 체결 및 파일럿 운영을 위한 모든 준비를 완료합니다.

### 세부 추진 일정

| 월차 | 추진 내용 | 담당 기관 | 완료 기준 |
| :--: | --------- | --------- | --------- |
| 1개월차 | 국방부·(주)티웨이브 MOU 체결 | 국방부 + (주)티웨이브 | MOU 서명 완료 |
| 1개월차 | 파일럿 부대 최종 선정 | 국방부 + 각 군 | 부대 확정 및 승인 |
| 2개월차 | 군 맞춤형 서비스 기능 개발 | (주)티웨이브 | 앱 업데이트 완료 |
| 2개월차 | 군 장병 금융 교육 콘텐츠 제작 | 국군복지단 + (주)티웨이브 | 교육 자료 완성 |
| 3개월차 | 부대 담당관 교육 | (주)티웨이브 | 담당관 교육 이수 |
| 3개월차 | 보안 심의 완료 | 각 군 보안 담당 | 앱 승인 완료 |
| 3개월차 | 홍보 자료 배포 | 국군복지단 | 포스터·안내문 배포 |

### 주요 준비 사항

**서비스 기술 준비 (티웨이브 담당)**
- 군 장병 전용 회원 가입 절차 간소화 (군 신분 인증 연계)
- 군 장병 전용 스테이지 옵션 추가 (복무 기간 연동 추천 기능)
- 복무 기간 설정 시 맞춤형 스테이지 자동 추천 기능 구현

**금융 교육 콘텐츠 준비 (공동 개발)**
- 입대 후 30분 이내 이수 가능한 기초 금융 교육 모듈 개발
- 빚투·불법 도박 예방 영상 콘텐츠 (5분 내외) 제작
- 아임인 이용 방법 안내 영상 제작

## 4.2.3 제2단계: 파일럿 운영 및 성과 검증 (2027년 상반기, 6개월)

### 목표

1개 사단급 부대에서 아임인을 시범 운영하고 실증 데이터를 확보합니다.

### 파일럿 규모 및 구성

| 항목 | 내용 |
| ---- | ---- |
| **시범 부대** | 1개 사단 (육군 기준) |
| **대상 장병 수** | 약 2,000명 (자율 참여) |
| **목표 가입률** | 50% 이상 (약 1,000명) |
| **운영 기간** | 6개월 |
| **스테이지 주력** | 5개월 및 7개월 단기 중심 |

### 파일럿 성과 측정 지표 (KPI)

| 지표 | 측정 방법 | 목표치 |
| ---- | --------- | :----: |
| **가입률** | 대상 장병 대비 가입 완료 비율 | 50% 이상 |
| **유지율** | 가입 후 3개월 지속 참여 비율 | 80% 이상 |
| **목돈 달성률** | 첫 스테이지 완료 비율 | 90% 이상 |
| **금융 사고 발생 건수** | 파일럿 부대 금융 징계 건수 | 전년 동기 대비 감소 |
| **만족도** | 이용자 대상 설문 조사 | 4.0/5.0 이상 |
| **순추천지수(NPS)** | 주변 장병 추천 의향 | 50 이상 |

## 4.2.4 제3단계: 중규모 확대 운영 (2027년 하반기, 6개월)

### 목표

파일럿 성과를 기반으로 5개 사단 수준으로 운영 규모를 확대하고 운영 체계를 고도화합니다.

| 항목 | 내용 |
| ---- | ---- |
| **확대 부대** | 각 군 1개 사단씩 5개 사단 |
| **대상 장병 수** | 약 1만 명 |
| **목표 가입률** | 60% 이상 (약 6,000명) |
| **추가 기능** | 금융 교육 수료 연동, 성과 보상 이벤트 |

### 확대 시 추가 보완 사항

- 파일럿에서 도출된 개선 의견을 전면 반영한 서비스 업데이트
- 부대별 담당관 네트워크 정식 구성
- 국군복지단 연계 금융 교육 정기 프로그램 운영 시작

---

## 4.2.5 제4단계: 전군 도입 및 지속 운영 (2028년 이후)

### 목표

전군 50만 장병을 대상으로 아임인 이용 환경을 구축하고, 군내 건전 저축 문화를 완전히 정착시킵니다.

| 항목 | 내용 |
| ---- | ---- |
| **대상** | 현역 병사 전원 (약 50만 명) |
| **목표 가입률** | 연간 신규 입대자의 70% 이상 |
| **연간 저축 규모 (추정)** | 가입자 30만 명 × 월 20만 원 × 12개월 = 약 7,200억 원 |
| **지속 운영 체계** | 연간 성과 평가 + 서비스 고도화 로드맵 지속 적용 |
