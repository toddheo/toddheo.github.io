# 6.3 주요 파트너십 및 공인 이력

**[2026-05-08]**

---

## 6.3.1 금융권 파트너십: 제도권 금융의 공식 인증

아임인이 단순한 P2P 서비스나 비공식 계모임과 근본적으로 다른 이유는, **대한민국 주요 제도권 금융기관이 공식 파트너로 검증**했다는 데 있습니다.

### ① 신한은행 파트너십

| 항목 | 내용 |
| ---- | ---- |
| **파트너십 유형** | 공식 제휴 파트너십 |
| **신한은행의 위상** | 국내 5대 시중은행 중 하나, 자산 규모 기준 상위 |
| **협력 내용** | 아임인 플랫폼 내 금융 서비스 연계, 공식 승인 서비스 지위 |
| **의미** | 신한은행의 엄격한 리스크 관리 기준을 통과한 공신력 확보 |

**국방부를 위한 해석:**
신한은행이 파트너십을 체결했다는 것은 아임인이 대한민국 최고 수준의 금융 기관의 심사를 통과했다는 의미입니다. 보안, 리스크 관리, 법규 준수, 서비스 안정성 모두 최고 수준으로 검증된 것입니다.

### ② 케이뱅크(KBank) 파트너십

| 항목 | 내용 |
| ---- | ---- |
| **파트너십 유형** | 공식 제휴 파트너십 |
| **케이뱅크의 위상** | 국내 최초 인터넷전문은행, 디지털 금융 혁신의 상징 |
| **협력 내용** | 디지털 금융 서비스 연계, 이용자 편의 확대 |
| **의미** | 디지털 금융 혁신성 및 기술 신뢰도 공인 |

## 6.3.2 정부·공공 기관 공인 이력

아임인은 민간 금융기관뿐 아니라 정부 및 공공기관으로부터도 복수의 공인을 받은 서비스입니다.

| 공인 기관/프로그램 | 내용 | 의미 |
| :---------------: | ---- | ---- |
| **신한퓨처스랩 10기** | 신한금융그룹 운영 혁신 핀테크 육성 프로그램 선정 | 금융 혁신성 및 성장 가능성 공인 |
| **창업도약패키지** | 중소벤처기업부 운영 창업 지원 프로그램 선정 | 정부 공식 인증 혁신 기업 |
| **서울 4대 미래성장산업 유망기업** | 서울시 선정 | 지방자치단체 수준의 공공 공인 |

---

## 6.3.3 파트너십이 군 도입에 주는 신뢰 근거

국방부를 비롯한 공공기관이 민간 서비스를 도입할 때 가장 중요하게 검토하는 것은 **서비스의 안정성과 공신력**입니다. 아임인의 파트너십 포트폴리오는 이 두 가지를 모두 충족합니다.

```
[아임인 공신력 검증 체계]

신한은행 ──┐
케이뱅크 ──┼── 금융권 최고 수준 심사 통과
           │
신한퓨처스랩 ──┐
창업도약패키지 ─┼── 정부·공공 공인 이력
서울시 유망기업 ─┘
           │
           ▼
   국방부 도입 신뢰 기반 완비
```

---

## 6.3.4 향후 파트너십 확장 계획 (군 도입 연계)

군 도입 성사 시 아임인은 군 특화 파트너십을 추가로 구성하겠습니다.

| 파트너십 유형 | 예상 파트너 | 역할 |
| :----------: | ---------- | ---- |
| **군인 전용 금융 연계** | 군인공제회, 국민은행 등 | 장병 전용 금융 혜택 추가 설계 |
| **금융 교육 콘텐츠 협력** | 금융감독원 금융교육원 | 공신력 있는 금융 교육 콘텐츠 공동 개발 |
| **복지 서비스 연계** | 국군복지단 | 기존 군 복지 서비스와 아임인 시너지 설계 |
