# 1.2 당면 과제: 급증하는 장병 빚투 실태 및 불법 도박 현황

**[2026-05-08]**

---

## 1.2.1 '빚투' 현상의 구조적 이해

'빚투'란 빚을 내어 투자하는 행위를 지칭하는 신조어입니다. 본래 성인 사회 전반에 걸친 문제이나, 군 복무 중인 장병들에게는 다음과 같은 이유로 더욱 빠르게, 더욱 깊게 침투하는 양상을 보입니다.

### 빚투 발생·심화 메커니즘 (군 특화 분석)

```
[Step 1] 급여 인상 → 가처분 소득 급증
           │
           ▼
[Step 2] SNS·유튜브를 통해 '단기 고수익' 투자 정보 접촉
           │  (병영 내 제한된 인터넷 환경에서 오히려 자극적 콘텐츠 집중 노출)
           ▼
[Step 3] 또래 장병 간 투자 경험 공유 → 집단 동조 압력 형성
           │  ("같은 부대 ○○는 코인으로 100만 원 벌었대")
           ▼
[Step 4] 소액 투자 시작 → 초기 수익 경험(또는 손실) → 몰입 심화
           │
           ▼
[Step 5] 급여 범위 초과 투자 욕구 → 군 내 사금융·선불 대여 이용
           │
           ▼
[Step 6] 복무 중 부채 발생 → 전역 후 신용 불량 위기
```

## 1.2.2 군내 금융 사고 실태 데이터

### 국내 20대 투기성 투자 현황 (군 장병 주연령층 기준)

| 투자 유형 | 20대 참여율 (2020년) | 20대 참여율 (2024년) | 증가율 |
| --------- | :------------------: | :------------------: | :----: |
| **국내 주식** | 15.2% | 41.3% | +171.7% |
| **암호화폐** | 9.8% | 31.0% | +216.3% |
| **해외 주식** | 3.1% | 18.7% | +503.2% |
| **파생상품·선물** | 1.2% | 7.4% | +516.7% |

> 출처: 한국거래소 투자자 현황, 금융위원회 실태조사 참고 재구성

20대 초반 장병들이 가장 활발하게 참여하는 암호화폐 및 해외 주식 투자는 **변동성이 극도로 크고, 투자 원금 전액 손실 가능성**이 상존하는 고위험 자산군입니다. 금융 지식이 부족한 상태에서의 참여는 사실상 투기에 가깝습니다.

## 1.2.3 빚투·불법 도박이 군 운영에 미치는 영향

금융 사고는 단순히 개인의 재무 문제에 머물지 않습니다. 군이라는 특수한 공동체에서 발생하는 금융 사고는 다음과 같이 **군 전투력과 조직 건전성에 직접적인 악영향**을 미칩니다.

### 군 전투력·조직 건전성에 미치는 파급 효과

| 피해 유형 | 개인 차원 | 부대 차원 | 국가 차원 |
| --------- | --------- | --------- | --------- |
| **재무적 손실** | 전역 후 신용 불량, 채무 | 사기 저하, 근무 태만 | 복지 재정 부담 증가 |
| **심리적 피해** | 우울, 불안, 극단적 선택 고려 | 동료 간 불신, 대인 갈등 | 군 이미지 실추 |
| **형사·군사적 제재** | 징계·영창·불명예 전역 | 지휘관 연대 책임 | 국방력 약화 |
| **가정·사회적 피해** | 가족 관계 파탄, 취업 불이익 | 불필요한 행정 처리 부담 | 사회 통합 비용 증가 |

## 1.2.4 취약 장병 유형별 분석

### 유형 ① 고위험 투자 유입형 (전체 위험 장병의 약 55% 추정)

- **프로파일:** 복무 전 주식 앱 이용 경험 보유, SNS에서 투자 콘텐츠 적극 소비
- **행동 패턴:** 월급 일부를 주식·코인에 자동 투자, 손실 발생 시 만회를 위해 투입 비중 증가
- **결과:** 전역 시 당초 계획한 시드머니 미확보, 경우에 따라 부채 보유

### 유형 ② 불법 도박 연루형 (전체 위험 장병의 약 30% 추정)

- **프로파일:** 병영 내 무료함을 달래기 위한 소액 도박 시작, 점차 몰입
- **행동 패턴:** 개인 메신저 기반의 도박방 참여, 선불 대여 이용
- **결과:** 형사처벌·징계, 가정 불화, 전역 후 채무 발생

### 유형 ③ 재무적 취약·위기형 (전체 위험 장병의 약 15% 추정)

- **프로파일:** 입대 전부터 경제적 어려움 보유, 가족의 부양 부담
- **행동 패턴:** 급여를 가족에게 전액 송금, 본인 비상금 전무, 개인 지출 극단적 절제
- **결과:** 심리적 절망감, 극단적 선택 고려, 사고 위험 최고조

## 1.2.5 '빚투'의 악순환 구조와 탈출 경로

현재 군 내에서 금융 사고에 처한 장병들이 이용할 수 있는 '건전한 탈출 경로'는 사실상 전무합니다.

```
┌────────────────────────────────────────────────────────────────┐
│                  현재의 악순환 구조                              │
│                                                                  │
│   급여 입금 → 운용처 없음 → 고위험 투기 → 손실                  │
│        ↑                                              │          │
│        └──────────────────────── 만회 심리 ←─────────┘          │
│                                                                  │
└────────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────────┐
│              아임인 도입 후 선순환 구조                           │
│                                                                  │
│   급여 입금 → 아임인 자동 납입 → 목표 달성 → 성취감              │
│        ↑                                              │          │
│        └──────────────── 새 목표 설정 ←───────────────┘          │
│                                                                  │
└────────────────────────────────────────────────────────────────┘
```

아임인이 군에 도입된다면, 장병들은 '투기'가 아닌 '목표 달성'이라는 건전한 방향으로 금융 행동을 재정립할 수 있습니다.
