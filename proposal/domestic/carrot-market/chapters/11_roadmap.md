# 2.7 추진 로드맵

---

## 1단계: 바터 트래픽 교환 (2026년 5~6월)

**목표:** 당근 유저에게 아임인을 알리고, 아임인 유저를 당근으로 유입

```
5월: 계약 체결, 배너 크리에이티브 제작
6월: 상호 배너 노출 시작
     → 당근 앱: 동네생활·당근페이 탭에 아임인 배너
     → 아임인 앱: 당근 제휴 배너
     → 전환 시 쿠폰 자동 지급 시스템 가동
```

**KPI:** 상호 전환율, 쿠폰 사용율, 신규 가입자 수

---

## 2단계: 당근페이 × 아임인 수령금 연동 (2026년 7~9월)

**목표:** 스테이지 만기 수령금을 당근머니로 전환 가능하게 하여 당근 생태계 내 소비 완결

```
7월: API 연동 기술 협의, 개발 착수
8월: 내부 테스트 및 QA
9월: 정식 오픈 — "수령금을 당근머니로 전환하면 3% 추가 지급"
```

**KPI:** 당근머니 전환 건수, 전환율, 당근 내 소비 증가액

---

## 3단계: 동네생활 모임통장 화이트라벨링 (2026년 10월 ~ 2027년 1분기)

**목표:** 당근 동네생활에서 모임 개설과 동시에 공동 저축 기능 이용 가능

```
10~11월: 법무 검토, UI/UX 설계, 브랜딩 확정
12월: 개발 및 내부 테스트
2027년 1월: 파일럿 운영 (특정 지역·모임 대상)
2027년 1분기: 전국 정식 출시
```

**KPI:** 모임통장 개설 수, 완주율, 동네생활 DAU 변화, 당근 생태계 내 소비 전환율
