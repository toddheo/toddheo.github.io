# 1.1 시장 배경 — 하이퍼로컬 플랫폼과 금융의 융합

## 당근마켓, 지역생활 플랫폼을 넘어 슈퍼앱으로

당근마켓은 2015년 중고거래 앱으로 출발해 10년 만에 국내 **월간 활성 사용자(MAU) 2,000만**, 글로벌 **3,000만** 사용자를 보유한 하이퍼로컬 슈퍼앱으로 성장했습니다. '당근하다'라는 신조어가 생길 정도로 한국인의 일상 깊숙이 자리잡은 플랫폼입니다.

| 항목 | 수치 | 비고 |
|------|------|------|
| 국내 MAU | **2,000만+** | |
| 글로벌 사용자 | **3,000만+** | |
| 2024년 연결 매출 | **1,892억 원** | YoY +48% |
| **2025년 Q1 매출** | **578억 원** | YoY **+38.4%** |
| **2025년 Q1 영업이익** | **164억 원** | YoY **+77.8%** |
| **2025년 연간 매출 예상** | **2,500억 원+** | 3분기 누적 이미 1,958억 |
| 누적 투자 유치 | **2,259억 원** | Series D, K-유니콘 |
| AI R&D 투자 (2025 상반기) | **406억 원** | 기술 고도화 집중 |
| 고용 인원 | **634명** | 직원 60%가 IT 개발자 |

> 📌 **2025년 연속 고성장** — 2025년 3분기 누적 매출이 2024년 연간 실적(1,892억)을 이미 초과. 연간 2,500억 돌파가 유력하며 IPO 준비 신호로 해석됩니다.

당근마켓은 중고거래를 기반으로 생활 서비스를 확장하고 있습니다.

| 서비스 | 내용 |

## 금융으로의 확장 — 당근페이의 가능성과 과제

2024년 6월 당근페이 출시는 당근마켓이 단순 거래 플랫폼을 넘어 **금융 생태계**를 구축하겠다는 선언입니다.

당근페이 주요 데이터:
- **12개월 재구매율: 51.3%** (2명 중 1명이 재결제)
- **연평균 구매 횟수: 12.8회**
- **주 사용자: 3040 남성** (성별·연령 집중 현상)
- **2024년 매출: 37억 원** (전체 매출의 **2%**)

당근페이의 재구매율과 사용 빈도는 높지만, 전체 수익에서 차지하는 비중이 매우 낮습니다. 광고 매출(1,888억)에 대한 과도한 의존이 수익 구조의 취약점입니다.

## 국내 소셜 금융 시장의 부상

ROSCA(Rotating Savings and Credit Association), 즉 디지털 계모임 시장이 빠르게 성장하고 있습니다.

- 한국의 전통적 '계' 문화 — 소그룹 신뢰 기반 공동 저축의 오랜 역사
- MZ세대의 공동 소비 트렌드 — 여행·캠핑·취미 목적의 그룹 자금 마련 수요 급증
- 핀테크 규제 환경 성숙 — 전자금융거래법 체계에서 ROSCA 운영이 법적으로 명확히 허용됨

아임인은 이 시장에서 **8년 이상** 운영 실적과 **3조 863억 원** 누적 거래액으로 독보적 1위를 유지하고 있습니다.

## 당근마켓 성공 요인 분석 — ERIS 모델 관점

학술 연구(문화산업연구, 2024)는 당근마켓의 성공을 ERIS 모델(창업가·자원·산업환경·전략) 로 분석합니다. 아임인과의 파트너십은 이 네 축 모두에서 당근의 과제를 해결합니다.

| ERIS 요소 | 당근마켓의 특성 | 아임인 협력 시너지 |
|-----------|-----------------|-------------------|
| **창업가(E)** | 하이퍼로컬·커뮤니티 신뢰 비전 — "대한민국 사회자본 회복" | 소그룹 신뢰 기반 ROSCA = 동일 비전의 금융적 구현 |
| **자원(R)** | IT 개발자 60%, AI 머신러닝, 2,270억 투자 / **수익모델 다각화 과제** | 아임인 ROSCA 플랫폼 즉시 화이트라벨링 → 개발 비용 0원 |
| **산업환경(I)** | 중장년층 이용 비율 **36%** (10명 중 4명이 3050+) — 경쟁사 대비 고령층 강점 | 아임인 핵심 사용자도 **3050+** — 완벽한 타겟 오버랩 |
| **전략(S)** | 하이퍼로컬(반경 6km), 매너온도 신뢰 지표, 직거래 → **수익 다각화가 최대 전략 과제** | 금융 서비스 라인업 추가 = 광고 99.8% 의존에서 탈피 |

> 📊 **핵심 데이터:** 당근마켓 이용자의 36%가 중장년층(2021 기준). 아임인의 주 이용층과 동일한 3050+ 코어 — 이는 상호 유저 전환에 있어 가장 강력한 근거입니다.

## 두 플랫폼의 DNA가 만나는 지점

| | 당근마켓 | 아임인 |
|---|---|---|
| **핵심 가치** | 이웃 간 신뢰 기반 커뮤니티 | 소그룹 신뢰 기반 공동 저축 |
| **사용자 행동** | 동네 이웃과 거래·소통 | 아는 사람끼리 함께 납입 |
| **플랫폼 특성** | 하이퍼로컬, 지역 인증 필수 | 그룹 내 상호 책임감 구조 |
| **공통 코어 타겟** | **3050 중장년층 36%** | **3050 중장년층 60%+** |
| **미충족 수요** | 이웃과 함께 자금 모으기 | 하이퍼로컬 유저 접점 확장 |

> **두 플랫폼의 결합은 기능적 통합이 아닌, 동일한 사용자 DNA의 자연스러운 연장입니다. 같은 이웃과 당근에서 거래하고, 아임인에서 함께 모읍니다.**
