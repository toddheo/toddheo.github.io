# 2.3 아임인 서비스 개요

---

## 아임인이란?

아임인(imin)은 전통 '계(契)' 문화를 디지털화한 소셜 금융(ROSCA) 플랫폼입니다. **2016년** 서비스 시작 이후 **8년 이상** 운영되며 **3조 863억 원**의 누적 거래액과 **60만 7,169명**의 참여자를 달성했습니다.

---

## 스테이지 운용 원리

아임인의 핵심 상품은 **'스테이지'**입니다. 스테이지는 N명이 N개월 동안 매월 동일한 금액을 납입하고, 순번에 따라 전체 적립금(N × 납입액)을 받아가는 구조입니다.

**9인 스테이지 예시 (월 20만 원, 9개월):**

```
참여 인원: 9명
납입 금액: 월 20만 원
스테이지 기간: 9개월
총 수령액: 180만 원 (+ 아임인 지원금)

1개월차: 1번 참여자 → 180만 원 수령 (나머지 8명 납입 지속)
2개월차: 2번 참여자 → 180만 원 수령
...
9개월차: 9번 참여자 → 180만 원 수령

→ 모든 참여자가 9개월 완주 시 동일 금액 수령
```

## 스테이지 구성 옵션

| 스테이지 | 인원 | 기간 | 납입(예시) | 수령액 |
|---------|------|------|-----------|--------|
| 5인 스테이지 | 5명 | 5개월 | 월 20만 원 | 100만 원 |
| 7인 스테이지 | 7명 | 7개월 | 월 15만 원 | 105만 원 |
| 9인 스테이지 | 9명 | 9개월 | 월 20만 원 | 180만 원 |
| 11인 스테이지 | 11명 | 11개월 | 월 30만 원 | 330만 원 |
| 13인 스테이지 | 13명 | 13개월 | 월 50만 원 | 650만 원 |

> **납입 범위:** 월 10만 원 ~ 최대 160만 원

## 주요 기능

| 기능 | 내용 |
|------|------|
| **스테이지 개설** | 목적·인원·금액을 설정하고 그룹 초대 |
| **자동 납입** | 약정일에 자동 이체, 납입 완료 알림 |
| **그룹 대시보드** | 납입 현황, 수령 순번, 잔여 기간 확인 |
| **수령금 관리** | 만기 수령금 계좌 이체 또는 당근머니 전환(연동 시) |
| **안심서비스** | 연체 발생 시 그룹 보호 자동 작동 |
