**[2026-05-14]**

# 2.8 파일럿 설계

---

## 파일럿 목표

- 당근 유저의 아임인 서비스 수용성 검증
- 전환율·완주율·당근머니 전환율 데이터 확보
- 화이트라벨링 본 출시 전 리스크 최소화

---

## 1단계 파일럿: 바터 배너 A/B 테스트 (2026년 6월)

| 항목 | 내용 |
|------|------|
| **대상** | 당근 동네생활 탭 방문자 |
| **방식** | A안: '동네 이웃이랑 함께 모아요' 배너 / B안: '아임인 스테이지 쿠폰 5만 원' 배너 |
| **기간** | 4주 |
| **측정 지표** | CTR(클릭률), 아임인 가입 전환율, 쿠폰 사용율 |

---

## 2단계 파일럿: 당근페이 연동 소프트 론칭 (2026년 9월)

| 항목 | 내용 |
|------|------|
| **대상** | 아임인 기존 스테이지 완주자 중 당근 유저 (자발적 참여) |
| **방식** | 수령금 → 당근머니 전환 옵션 제공, 3% 보너스 지급 |
| **기간** | 3개월 |
| **측정 지표** | 당근머니 전환 건수, 전환 후 당근 내 소비 패턴, 재참여율 |

---

## 3단계 파일럿: 동네생활 모임통장 지역 파일럿 (2027년 1월)

| 항목 | 내용 |
|------|------|
| **대상** | 서울 3~5개 구 동네생활 모임 운영자 |
| **방식** | 모임 개설 시 '모임 자금 모으기' 기능 제안, 자발적 참여 |
| **기간** | 3개월 (1개 스테이지 완주 기준) |
| **측정 지표** | 모임통장 개설 수, 완주율, 참여자 만족도, 당근머니 전환율 |

---

## 성공 기준

| 지표 | 목표값 |
|------|--------|
| 바터 배너 CTR | 2% 이상 |
| 아임인 신규 가입 전환율 | 15% 이상 |
| 스테이지 완주율 | 85% 이상 |
| 당근머니 전환율 | 완주자의 30% 이상 |
| 파일럿 NPS | 70 이상 |
