# 2.4 당근 유저 맞춤 시나리오

아임인 서비스는 만 19세 이상 성인만 이용 가능합니다.

---

## 시나리오 A. 동네 이웃 모임형 — 정기 모임 자금을 함께 모은다

**상황:** 당근 동네생활에서 만난 동네 이웃 5명(성인), 매달 함께하는 동네 맛집 탐방 모임. 매번 회비 걷기가 번거롭고, 큰 식사 이벤트 때 목돈이 필요하지만 미리 모을 방법이 없다.

| 항목 | 내용 |
|------|------|
| **스테이지 구성** | 5인 스테이지, 월 10만 원, 5개월 |
| **수령액** | 50만 원 |
| **아임인 지원금** | +5만 원 (1인당) |
| **실질 모임 자금** | **55만 원** |
| **적합 용도** | 동네 특별 식사 모임, 당근 지역 행사 참여비 |

> "매달 10만 원씩 당근 이웃이랑 모았더니 55만 원이 됐어요. 회비 걷는 번거로움도 없고 다들 빠지지 않았어요."

## 시나리오 B. 동네 여행 모임형 — 이웃과 함께 여행 자금을 미리 준비한다

**상황:** 당근 동네생활 '우리 동네 여행 모임' 성인 회원 7명. 매년 한 번 단체 국내 여행. 당일 정산이 복잡하고 미리 모을 방법이 없었다.

| 항목 | 내용 |
|------|------|
| **스테이지 구성** | 7인 스테이지, 월 15만 원, 7개월 |
| **수령액** | 105만 원 |
| **아임인 지원금** | +6만 원 (1인당) |
| **실질 여행 자금** | **111만 원** |
| **적합 용도** | 국내 여행 숙박·교통비, 당근머니 전환 후 당근 내 여행 용품 구매 |

> "7명이서 7개월 함께 모으니까 절대 빠지는 사람 없이 다 완주했어요. 여행 자금 걱정이 사라졌어요."

## 시나리오 C. 당근알바 수입 목돈 마련형 — 불규칙 수입을 목돈으로 전환한다

**상황:** 당근알바로 주 2~3회 단기 아르바이트를 하는 성인 9명. 월수입이 일정하지 않아 목돈 마련이 어렵고, 개인 저축은 작심삼일로 끝나는 경우가 많다.

| 항목 | 내용 |
|------|------|
| **스테이지 구성** | 9인 스테이지, 월 20만 원, 9개월 |
| **수령액** | 180만 원 |
| **아임인 지원금** | +7만 원 (1인당) |
| **실질 목돈** | **187만 원** |
| **적합 용도** | 비상금 마련, 전자제품 구입, 자기계발비 |

> "당근알바 수입 일부를 9명이서 9개월 모으니까 187만 원이 됐어요. 혼자 했으면 중간에 썼을 돈인데."

## 시나리오 D. 동네 취미 모임 장비 구입형 — 공동 장비를 함께 마련한다

**상황:** 당근 동네생활 '우리 동네 테니스 모임' 성인 회원 5명. 공동으로 쓸 테니스 라켓 세트와 볼 머신(목표 100만 원) 구입 계획.

| 항목 | 내용 |
|------|------|
| **스테이지 구성** | 5인 스테이지, 월 20만 원, 5개월 |
| **수령액** | 100만 원 |
| **아임인 지원금** | +5만 원 (1인당) |
| **실질 구입 자금** | **105만 원** |
| **적합 용도** | 당근 중고 장비 구매 또는 당근머니 전환 후 구입 |

> "5명이서 5개월 모으니까 장비 살 목돈이 생겼고, 지원금 5만 원으로 공 추가 구입했어요."
