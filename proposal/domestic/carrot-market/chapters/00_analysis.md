## 2. 당근마켓의 전략적 포지셔닝 — ERIS 모델 분석

ERIS 모델(Sandberg & Hofer, 한정화 2003)은 창업기업의 성과를 **창업가(E) · 자원(R) · 산업환경(I) · 전략(S)** 4요소로 설명합니다. 학계는 당근마켓의 성공 요인을 이 모델로 분석했으며(문화산업연구 제24권 제3호, 2024), 이는 아임인과의 협력 포인트를 정교하게 설계하는 데 중요한 근거입니다.

### 창업가(Entrepreneur) — 하이퍼로컬 신뢰 회복 비전

김용현·김재현 공동대표는 "대한민국 사회자본이 167개국 중 107위"라는 현실에서 출발해 **동네 신뢰 회복**을 핵심 사명으로 삼았습니다. 판교장터(사내 중고거래) → 전국 하이퍼로컬로 확장한 경험은 "같은 공간을 공유하는 사람끼리는 신뢰가 높다"는 원칙을 실증했습니다.

**아임인 연결 포인트:** 아임인의 ROSCA도 본질적으로 신뢰 기반 소그룹 금융. 당근의 "동네 이웃 신뢰"와 아임인의 "같이 모으는 사람 신뢰"는 같은 DNA입니다.

### 자원(Resources) — IT 기술력과 수익 다각화 과제

- 전체 직원(634명) 중 **60%가 IT 개발자**
- AI 머신러닝으로 불법 게시물 90% 자동 차단
- 2025 상반기 R&D 투자 **406억 원** — 기술 고도화 집중
- **그러나 금융 상품 라인업은 빈약** — 당근페이 37억(2%)에 그침

**아임인 연결 포인트:** 아임인 ROSCA 화이트라벨링은 당근이 **개발 비용 0원**으로 금융 상품 라인업을 즉시 추가하는 경로. 당근의 기술력은 유저 경험 최적화에 집중 가능.

### 산업환경(Industry) — 하이퍼로컬 1위, 중장년층 강점

- 중고거래 앱 단독 이용률 **81%** — 당근 유저의 충성도 압도적
- **중장년층(3050) 이용 비율 36%** — 경쟁사(번개장터 10~20대 54%) 대비 고령층 강점
- 직거래·지역 기반 → 공동체 소속감 형성 → 락인 효과 강함

**아임인 연결 포인트:** 아임인의 핵심 사용자도 **3050 중장년층(60%+)**. 두 플랫폼의 유저 프로필이 가장 높게 겹치는 구간 — 상호 전환 마케팅 효율 극대화.

### 전략(Strategy) — 하이퍼로컬 선점, 수익 다각화가 과제

- 반경 6km 지역 인증 + 매너온도 → 동네 신뢰 시스템 구축
- 알바·부동산·중고차 버티컬 확장 중
- **광고 99.8% 의존 → 수익 다각화가 최대 전략 과제**
- 학계 분석: "수익 증대를 위한 노력이 당근의 커뮤니티 정체성과 충돌할 수 있다는 비판"

**아임인 연결 포인트:** 아임인 파트너십은 당근이 자체 금융 상품 출시 없이 **외부 파트너십**으로 수익을 추가하는 방식 → 커뮤니티 정체성 훼손 없이 수익 다각화 가능.

## 3. 핵심 협력 포인트 — 맞춤형 설계

아임인과의 파트너십이 당근마켓에게 갖는 전략적 의미를 3가지 핵심 포인트로 정리합니다.

### 협력 포인트 1. 수익 다각화의 가장 빠른 경로

당근페이 매출(37억)을 끌어올리는 데 있어 자체 금융 상품 개발은 수개월의 법무 검토와 개발이 필요합니다. 반면 아임인과의 바터 제휴는 **2주 내 집행**, 수령금 연동은 **API 연동 1~2개월**이면 가능합니다.

```
당근 자체 금융상품 개발 → 6개월+ 법무·개발 (리스크 있음)
아임인 화이트라벨링 →    3~6개월 (개발 비용 0, 법무 리스크 아임인 부담)
아임인 바터 제휴 →        2주 (즉시 실행)
```

### 협력 포인트 2. 3050 중장년 코어 타겟 완전 공략

| 구분 | 당근 3050 이용률 | 아임인 3050 이용률 | 시너지 |
|------|----------------|------------------|--------|
| 비율 | **36%** | **60%+** | 중장년 금융·커뮤니티 원스톱 |
| 니즈 | 동네 이웃과 소통·거래 | 목돈 마련·공동 저축 | 이웃과 함께 모으기 |
| 이탈 위험 | 금융 기능 부재 시 카카오로 이탈 | 오프라인 접점 부재 | 상호 보완 |

### 협력 포인트 3. 당근 생태계 내 금융 완결 루프

현재 당근 유저는 거래 후 자금 운용을 위해 카카오페이, 토스 등 외부 앱으로 이탈합니다. 아임인 연동은 이 이탈을 막는 **당근 생태계 내 금융 완결 루프**를 만듭니다.

```
[당근 중고거래] → 수익 발생
    → [아임인 스테이지] 납입 (당근 앱 내)
    → [목돈 수령] → [당근머니 전환] 보너스 3~5%
    → [당근 내 소비 완결]
```

## 4. 경쟁사 동향과 선점 압박

| 경쟁사 | 하이퍼로컬 + 금융 시도 | 위협 수준 |
|--------|----------------------|---------|
| 카카오페이 | 모임통장, 청구서 기능 강화 | 🔴 직접 경쟁 |
| 토스 | 저금통·목표 저축, 커뮤니티 없음 | 🟡 간접 경쟁 |
| 네이버 | 이웃톡·중고거래 결합 시도 | 🟡 간접 경쟁 |
| **아임인 + 당근** | **하이퍼로컬 × 소셜 금융 결합** | ✅ 선점 기회 |

> 한국 시장에서 **하이퍼로컬 커뮤니티 + 소셜 금융(ROSCA)**을 결합한 서비스는 아직 존재하지 않습니다. 선점 기업은 6~12개월 리드타임을 확보합니다.
