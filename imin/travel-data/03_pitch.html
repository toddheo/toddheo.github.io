Problem

여행업계가 풀지 못한 문제

기존 여행 마케팅은 "관심 있는 사람"을 겨냥합니다. 하지만 관심과 예산 확보는 다릅니다.

❌ 기존 여행 마케팅의 한계

  • 검색 기반 타겟팅 — 관심만 있고 예산은 없는 사용자
  • 계절·이벤트 기반 프로모션 — 타이밍 미스 다수
  • 광고 클릭 후 구매 전환율 1~3% 수준
  • 여행 의도는 있으나 예산 확보 여부 알 수 없음
VS

✅ 아임인 데이터 파트너십

  • 예산 확보가 확정된 사용자만 타겟
  • 목돈 수령 1~4주 전 — 최적 구매 시점 정확 포착
  • 여행 목적을 본인이 직접 선언한 고인텐트 사용자
  • 수령 금액 구간 기반 맞춤 상품 레코멘데이션 가능
Solution

아임인이 드리는 것

📅

수령 시점 데이터

사용자가 목돈을 수령하는 연·월을 사전에 알 수 있습니다. 수령 1개월 전 마케팅이 가능합니다.

💰

예산 구간 데이터

100만원대 / 200만원대 / 300만원 이상 구간으로 예산 규모를 파악하고 최적 상품을 제안하세요.

✈️

확정된 여행 인텐트

사용자가 직접 '여행'을 목적으로 스테이지에 참여한 고인텐트 세그먼트입니다. 가장 확실한 신호입니다.
